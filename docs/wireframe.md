# Star Atlas — MVP 와이어프레임

> MVP 범위: 현재 위치/시각 기반 3D 밤하늘 렌더링  
> 총 화면 수: **3개**

---

## 화면 전환 흐름

```
┌─────────────────┐     권한 허용      ┌─────────────────┐     로드 완료     ┌─────────────────┐
│                 │ ──────────────────▶ │                 │ ────────────────▶ │                 │
│  [01] 권한 요청  │                    │  [02] 로딩       │                   │  [03] 메인 하늘  │
│                 │ ◀────────────────── │                 │                   │      뷰         │
└─────────────────┘     권한 거부       └─────────────────┘                   └─────────────────┘
        ▲                                                                              │
        │                          권한 설정 후 재시도                                  │
        └──────────────────────────────────────────────────────────────────────────────┘
                                    (설정 앱 이동 후 돌아올 때)
```

---

## [01] 권한 요청 화면

**목적**: GPS 위치 권한 + 필요시 설명 제공  
**진입 조건**: 앱 최초 실행 또는 권한이 없는 상태

```
┌─────────────────────────────────┐
│                                 │  ← 상태바 (시스템)
│                                 │
│                                 │
│                                 │
│          ✦  ·  ✦               │
│        ·    ✦    ·             │
│      ✦   · ✦ ·   ✦            │  ← 별 일러스트 (정적 이미지)
│        ·    ✦    ·             │
│          ✦  ·  ✦               │
│                                 │
│                                 │
│        Star Atlas               │  ← 앱 타이틀 (center, large)
│                                 │
│   지금 이 순간, 당신 위치에서    │  ← 서브타이틀 (center, medium)
│   보이는 실제 밤하늘             │
│                                 │
│                                 │
│  ┌───────────────────────────┐  │
│  │   📍 위치 접근 허용하기    │  │  ← Primary CTA 버튼
│  └───────────────────────────┘  │
│                                 │
│    위치 정보는 하늘 계산에만     │  ← 안내 텍스트 (caption, center)
│    사용되며 저장되지 않습니다    │
│                                 │
└─────────────────────────────────┘
```

**인터랙션**
- `위치 접근 허용하기` 탭 → 시스템 권한 다이얼로그 표시
  - 허용 → [02] 로딩 화면으로 이동
  - 거부 → 버튼이 `설정에서 권한 허용하기`로 변경, 탭 시 설정 앱으로 이동

## [02] 로딩 화면

**목적**: GPS 수집 + 별 카탈로그 로드 + 좌표 계산 대기  
**예상 소요 시간**: 1~3초

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│          ◉                     │  ← 애니메이션 스피너
│        ·   ·                   │    (별 모양이 천천히 회전)
│                                 │
│                                 │
│      하늘을 불러오는 중...       │  ← 상태 텍스트 (center)
│                                 │
│   ████████████░░░░░░░░░░░░░    │  ← 프로그레스 바
│                                 │
│   📍 서울 · 2026.06.27 23:14   │  ← 수집된 위치·시각 표시
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
└─────────────────────────────────┘
```

**단계별 상태 텍스트**
```
1단계: "위치를 확인하는 중..."
2단계: "별 목록을 불러오는 중..."  (약 1700개 필터링)
3단계: "하늘을 계산하는 중..."    (좌표 변환)
```

**예외 처리**
- GPS 타임아웃(5초 초과) → "위치를 가져올 수 없습니다" 토스트 + [01]로 복귀
- 로드 실패 → "다시 시도" 버튼 표시

## [03] 메인 하늘 뷰 화면

**목적**: 3D 밤하늘 탐색 (MVP 핵심 화면)  
**진입 조건**: 로딩 완료 후

```
┌─────────────────────────────────┐
│  ☰              23:14   📍서울 │  ← 상단 정보 바 (반투명)
├─────────────────────────────────┤
│  ·    ✦       ·    ✦    ·     │
│     ✦    ·  ✦         ✦      │
│  ·    ·       ✦    ·    ·     │
│     ✦    ✦        ✦    ✦     │
│  · ·   ✦    ·  ·       ·  ✦  │  ← 3D 하늘 뷰 (전체 화면)
│    ✦        ✦    ✦    ·      │    별 밝기 = 등급(magnitude) 반영
│  ·    ✦  ·    ✦    ·    ✦   │    드래그: 구체 회전
│     ·    ✦  ·    ✦   ·   ·  │    핀치:   확대/축소
│  ✦    ·       ✦    ✦    ·   │
│     ✦    ✦       ·    ✦  ✦  │
│  ·    ✦   ·   ✦    ·    ·   │
│     ·       ✦    ✦    ✦  ·  │
│  ✦    ·  ✦    ·    ✦    ·   │
│     ✦    ·   ✦    ·    ✦    │
├─────────────────────────────────┤
│  ───────────────  지평선       │  ← 지평선 표시선 (수평 기준)
│                                 │
│  ↑ 북   ← 서     동 →   남 ↓  │  ← 방위 힌트 (현재 바라보는 방향 기준)
│                                 │
│  [ 현재 하늘로 돌아오기 ]       │  ← 뷰가 틀어졌을 때만 등장 (floating)
└─────────────────────────────────┘
```

**UI 요소 상세**

```
┌─ 상단 정보 바 ──────────────────┐
│  ☰   메뉴 아이콘 (향후 확장용)  │
│       23:14   현재 시각        │
│       📍 서울  현재 위치명      │
└─────────────────────────────────┘

┌─ 3D 뷰 제스처 ─────────────────┐
│  한 손가락 드래그  → 구체 회전  │
│  두 손가락 핀치    → 줌 인/아웃 │
│  두 손가락 드래그  → 시점 이동  │
│  별 탭 (MVP 外)   → 별 정보 팝업│
└─────────────────────────────────┘

┌─ 별 표현 규칙 ──────────────────┐
│  magnitude 1 미만  → 크고 밝게  │
│  magnitude 1~3     → 중간       │
│  magnitude 3~5     → 작고 희미  │
│  지평선 아래 별    → 렌더링 제외 │
└─────────────────────────────────┘
```

**"현재 하늘로 돌아오기" 버튼**
- 초기 진입 시 숨김
- 사용자가 하늘을 돌려 현재 천정 방향에서 15° 이상 벗어나면 등장
- 탭 시 애니메이션으로 현재 천정 방향으로 카메라 복귀
